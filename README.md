# Instructions to create Assignment.docx

This repository contains all materials required to produce the final Microsoft Word document Assignment.docx for the KC-PLUS assignment.

Files in this commit:
- Assignment.docx.txt — The complete, Word-ready assignment text. Paste this content into a new Word document and apply the recommended styles.
- DFD_KCPLUS.svg — Vector Data Flow Diagram (black/gray). Insert into Word as a picture (Insert > Pictures > This Device...). Word will preserve vector quality for resizing and printing.
- ERD_KCPLUS.svg — Vector Entity Relationship Diagram (black/gray). Insert into Word after the ERD section.

Recommended Word formatting steps:
1. Open Microsoft Word and create a new blank document.
2. Paste the contents of Assignment.docx.txt into the document.
3. Apply Times New Roman, 12pt to body text. Use Heading 1 (14–16pt bold) for main headings.
4. Place the cursor after section 6 and insert DFD_KCPLUS.svg (Insert > Pictures > This Device...).
5. Place the cursor after section 7 and insert ERD_KCPLUS.svg.
6. Adjust sizing of the images to fit the page (approx. 80–90% width). Because the images are SVG they will scale without loss.
7. Save the document as Assignment.docx (File > Save As).

If you would like, I can also create a pull request branch with an embedded .docx (rasterized diagrams) instead. Request that if you prefer a single ready .docx file in the repo.
